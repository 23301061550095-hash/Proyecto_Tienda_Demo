Tienda Demo – Proyecto HTML, CSS y JavaScript

Este proyecto es una tienda web sencilla creada con HTML, CSS y JavaScript. Incluye un listado de productos, un buscador, filtros por marca, ordenamiento y un carrito funcional con la opción de comprar.

Características principales
1. Visualización de productos

Los productos se generan dinámicamente desde un arreglo en JavaScript.
Cada tarjeta muestra:

Imagen del producto

Nombre

Marca

Precio

Botón para agregar al carrito

2. Buscador de productos

El usuario puede buscar por nombre mediante un campo de texto.
La búsqueda filtra en tiempo real y actualiza la lista mostrada.

3. Filtro por marca

Incluye un menú desplegable con las marcas disponibles.
Al seleccionar una marca, la tienda muestra únicamente los productos que pertenecen a esa categoría.

4. Ordenar productos

Ofrece diferentes formas de ordenar la lista:

Precio: menor a mayor

Precio: mayor a menor

Nombre: A a Z

Nombre: Z a A

5. Carrito de compras

El proyecto incluye un carrito completamente funcional:

Agregar productos desde el listado

Ver el contenido del carrito mediante un modal o panel lateral

Mostrar la cantidad total de artículos

Mostrar el total a pagar

Botón para completar la compra

6. Botón de compra

Dentro del carrito se muestra un botón para finalizar la compra.
Al presionarlo:

Se limpia el carrito

Se muestra un mensaje de confirmación

7. Código simple y entendible

El proyecto está hecho para ser claro y fácil de modificar.
Solo utiliza HTML, CSS y JavaScript puro, sin librerías externas.
